# 市场分析 · Market Analysis

> 本文是产品决策的依据。结论先行:**autofill 是红海,不要再做一个批量投递工具;真正服务不足的是"面试情报采集"和"可积累的个人职业上下文层"。**

## 一、市场现状:批量投递层已饱和,转化率崩塌

数十家工具已在做多平台自动投递,大致分三派:

- **完全自动轰炸式**:LazyApply、Sonara — 用户评价差(LazyApply 长期 ~1.9–2.1 星),"浪费钱"是高频反馈。
- **半自动自填**:Simplify、SpeedyApply、FastApply — 用户自己点投递,体验更受认可。
- **策展型 agent**:Jobright、ApplyIQ — 质量优先,设匹配阈值才触发。

**关键数据点**:

- 通过 LinkedIn 提交的申请激增,平台一度记录到每分钟上万份申请。
- 招聘方收到的申请量是几年前的数倍,Workday/Greenhouse/Lever 上线了重复申请标记。
- **体量腐蚀质量**:有平台数据显示,投 11–20 份的面试转化率约 9%,投 100+ 份只有约 2.6%,不足前者三分之一。
- 大规模投递厂商自己也承认:入门级岗位的面试率仅 1–4%。

## 二、竞争格局扫描

### 全球 / 英文市场

| 产品 | 类型 | 模式 | 局限 |
|------|------|------|------|
| **Simplify** | 浏览器扩展 | 免费 autofill,100+ ATS | 纯规则,陌生表单易失灵 |
| **Jobright** | Agent 原生 | VC 支持,52 万+用户,质量优先 | 仅美国,无面试准备 |
| **LazyApply** | 批量轰炸 | 一次性付费,1500/天 | 口碑差,封号风险 |
| **SpeedyApply** | 浏览器扩展 | autofill + AI 字段 | 同 Simplify 局限 |
| **Teal / Jobscan** | 简历优化 | 追踪 + ATS 扫描 | 无投递,UI 偏旧 |
| **Final Round AI** | 面试 copilot | 实时面试辅助 | 独立工具,与简历脱节,价高 |

### 中文市场

- **平台**:Boss直聘(占主导)、智联、前程无忧、猎聘、拉勾。Boss直聘自己已上线 AI 功能(mock 面试、简历指导、面试题"潜台词"解码)。
- **自动投递工具**:主要散落在 GitHub 开源脚本(如 `loks666/get_jobs`、`boss-agent-cli`)和灰色市场 Chrome 扩展,**没有像美国那样的融资消费产品**。
- Boss直聘对新对话有每日上限(~100),是对自动化的硬节流。

### 远程市场

- Wellfound(前 AngelList Talent)、Otta、We Work Remotely、RemoteOK。
- 痛点:大量申请未被查看。

## 三、关键洞察:gap 在哪里

把现有产品摊开看:

```
简历优化     ████████████  成熟(Jobscan/Teal/Rezi)
投递追踪     ████████████  成熟(Huntr/Teal/Careerflow)
autofill     ████████████  红海(Simplify/Jobright/SpeedyApply,免费)
面试 copilot ████████      有专项(Final Round AI),但与简历脱节
─────────────────────────────────────────────
面试情报采集  ███           服务不足 ← 机会
个人职业上下文 ██            几乎空白 ← 最大机会
```

**没有人在做的事**:

1. **结构化的个人面试情报** — 每场面试被问了什么、哪个公司、哪个阶段、你怎么答的、结果如何。这份数据随使用积累,是 Glassdoor 做不到的"你自己的面试库"。
2. **持久的个人职业上下文层** — 把简历、投递、面试、偏好统一成一个不断变富的对象,让每次定制和准备都更省力、更精准。

## 四、为什么不做批量投递

1. **市场已死**:转化率 2.6%,雇主反感,口碑普遍差(1.9 星)。
2. **危及用户**:服务器端代投会封号,责任落在用户头上。
3. **维护地狱**:对抗平台反爬是无尽的军备竞赛,solo / 小团队吃不消。
4. **会被商品化**:通用 agent 浏览器(如各类 AI browser)正逐步能免费做 ad-hoc 投递。

## 五、Relay 的定位

> **不在自动化体量上竞争,而是拥有结构化、可积累、便携的个人职业 + 面试数据。**

切入点:**质量优先的客户端投递(零封号)+ 面试情报采集 + 数据飞轮**,包裹在"AI 先做、用户后审"的极简体验里。

- vs Simplify/SpeedyApply(纯规则 autofill):我们用 LLM 处理任何表单 + 开放题,质量更高。
- vs Jobright(也质量优先):我们加了面试准备 + 趋势 + 数据飞轮,是完整 journey 而非单点。
- vs Final Round AI(面试独立工具):我们把面试和简历/投递打通。
- vs LazyApply(批量):我们不轰炸,零封号,质量优先。

## 六、商业模式

- **B2C 订阅**($15–25/月)为起点。简历管理 + 面试准备 + 趋势报告。
- **可选 BYO key** 把 LLM 成本归零,只收订阅费。
- **第二曲线**:向训练营 / 大学 / 职业教练出售结构化面试数据集或个人上下文层(B2B2C)。
- **增长引擎**:基于自有(征同意、聚合)数据发布求职趋势报告。

成本极友好:混合方案下每次投递 LLM 成本约 $0.003,100 投递/月 = $0.30/用户,订阅 $15 毛利 ~98%。
