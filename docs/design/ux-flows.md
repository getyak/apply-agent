# UX 流程 · UX Flows

> 可交互原型见 [`assets/product-ui-prototype.html`](../assets/product-ui-prototype.html)(5 屏可点击切换)。

整个产品是一个最短闭环:**启动 → 发现 → 投递 → 追踪 → 准备**,每环卡在"3 次点击以内"。

## 屏 1:启动(Onboarding · 一步)

```
┌──────────────────────────────────────┐
│         一步开始                        │
│   把简历放进来,剩下的交给我。            │
│                                        │
│   ┌──────────────────────────┐        │
│   │  ↑  拖入简历,或点击选择    │        │
│   │     PDF · Word · 10MB     │        │
│   └──────────────────────────┘        │
│         或者                           │
│   [用 LinkedIn 导入] [粘贴简历文字]      │
│                                        │
│   密码永不存储 · 数据可全程留在本地       │
└──────────────────────────────────────┘
           ↓ AI 解析(~5s)
   ✓ 3 段工作经历 · 14 项技能
   ✓ 识别方向:后端/全栈工程师
   ✓ 已匹配 28 个在招职位
```

唯一动作:上传简历。一个字段都不用填。

```
┌──────────────────────────────────────────────┐
│ ← [G] Senior Backend Engineer  ● 全部备好待审核 │
├────────────────────┬─────────────────────────┤
│ ◆ 为什么匹配 94%     │ 已为这个职位定制好         │
│ ✓ 5 年后端经验       │ 姓名: Xinwei Xiong        │
│ ✓ Go/K8s/分布式全覆盖 │ 邮箱: ...                 │
│ ✓ 开源贡献加分        │                          │
│                     │ 简历摘要 [AI 已调整]       │  橙
│ 职位要求             │ ┌──────────────────────┐ │
│ 必备:Go·分布式·K8s   │ │ 资深后端工程师...[编辑] │ │
│ 加分:CI/CD·调优      │ └──────────────────────┘ │
│                     │ 为什么加入? [AI 生成·草稿] │  橙
│                     │ ┌──────────────────────┐ │
│                     │ │ 我长期专注...  [编辑]   │ │
│                     │ └──────────────────────┘ │
├────────────────────┴─────────────────────────┤
│ 已填 9 字段·2 项 AI 生成   [保存草稿][审核无误,投递→]│
└──────────────────────────────────────────────┘
```

- 左:为什么匹配(给信心)+ JD
- 右:填好的内容,AI 部分橙色高亮
- 用户扫一眼、改几个字、投

## 屏 5:追踪(自然衔接)

```
┌──────────────────────────────────────────────┐
│ 我的投递              本月 12 份·3 场面试        │
│ 投出去就自动进来。有面试的会浮出"准备面试"。     │
├──────────────┬──────────────┬─────────────────┤
│ 已投递 6      │ 面试中 3      │ 有结果 1         │
│ [G] Google    │ [S] Stripe    │ [V] Vercel       │
│  2 天前·等待   │  周五 14:00   │  ✓ 进入终面       │
│              │ [◷ 准备面试]   │                  │
│ [L] Linear    │ [N] Notion    │                  │
│  4 天前·等待   │ [◷ 准备面试]   │                  │
└──────────────┴──────────────┴─────────────────┘
[0 配置] [3 次点击投递] [AI 先做] [零封号]
```

- 投递自动进看板(已投/面试中/有结果)
- 有面试的卡片浮出"准备面试"——上下文浮现,而非菜单导航

## 闭环串联

```
启动(上传简历)
  → 发现(每日简报,AI 已匹配排序)
    → 投递(AI 备好包,用户审核后本地提交)
      → 追踪(自动进看板)
        → 准备(有面试时浮出按钮)
          → 沉淀(全程进数据飞轮)→ 回到发现,更懂你
```
